# to2.at - URL 단축 서비스

## 개요

긴 URL을 짧게 줄여주는 서비스입니다.

- **도메인**: `to2.at`
- **인증**: 로그인 필수

## 핵심 기능

### URL 단축 시 2개 링크 생성

하나의 URL을 단축하면 **전체 공간**과 **개인 공간** 2개의 단축 URL이 생성됩니다.

| 구분 | 형식 | 예시 | 특징 |
|------|------|------|------|
| **전체 공간** | `/{global-seq}` | `to2.at/1` | 1년 미사용 시 자동 삭제 |
| **개인 공간** | `/{member-id}/{personal-seq}` | `to2.at/5/1` | 영구 보관 |

### 예시

회원 ID가 5인 사용자가 첫 번째 URL을 단축하면:
```
전체 공간: to2.at/1
개인 공간: to2.at/5/1
```

회원 ID가 5인 사용자가 두 번째 URL을 단축하면:
```
전체 공간: to2.at/2
개인 공간: to2.at/5/2
```
